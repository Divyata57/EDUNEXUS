# 🎓 EduNexus — Smart Campus Hub

EduNexus is a responsive, low-latency Smart Campus management application designed to seamlessly bridge operational pipelines between **Faculty Admin Terminals** and **Personalized Student Dashboards**. Engineered as a high-fidelity Relational Database simulation. this platform streamlines roster workflows, dynamic notifications, and real-time class activity tracking.

---

## 🚀 Core Features Deployed

### 🔐 1. Dynamic Role-Based Authentication
* **Gateway Matrix Security:** Implements absolute route protection mechanisms for different access profiles.
* **Faculty Admin Account:** Pre-seeded core privileges accessible via terminal ID `FAC2026` with password encryption fallback checks.
* **Student Registry Pool:** Auto-maps unique accounts for pre-loaded active sessions:
  * **Sharin:** Roll No: `254780` (Matted to Section A)
  * **Ritika:** Roll No: `254789` (Matted to Section B)

### 📊 2. Section-Pattern Attendance Management
* **Roster Data Sheets:** Faculty members can select a target section (Section A or Section B) to instantly render the filtered roster mapping list.
* **One-Click Action Controllers:** Integrated binary **`P` (Present)** and **`A` (Absent)** quick-triggers to adjust records dynamically.
* **Automated Safety Warnings:** Triggers custom pop-up validation alerts for students if their attendance falls below the mandatory **75% threshold**.

### 🔔 3. Real-Time Broadcast Sync Engine
* **Instant Client Notification Node:** When the faculty admin marks attendance, an automated notification block containing timestamp arrays is generated.
* **Reactive Alerts:** Upon routing or loading the dashboard, the targeted student’s node immediately captures unread updates and displays live interactive browser alerts.

### 📅 4. Integrated Smart Resource Hubs
* **Notice Board Matrix:** Clean component displaying real-time urgent campus events (e.g., *Tomorrow's BTech RAI Orientation Ceremony in Seminar Hall-3*).
* **Weekly Class Schedule Grid:** Comprehensive Monday-to-Friday timeline grid charting exact module slots and lab timelines.
* **Assignment Deliverable Tracker:** State management tracking task statuses (`Completed` / `Pending`) across various courses like Python Programming and Instrumentation.
