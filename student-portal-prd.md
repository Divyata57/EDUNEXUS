# Product Requirement Document (PRD) — Student Portal

## 1. Project Overview & Objectives
Goal is to build a unified, responsive Student Management and Productivity Portal for academic schedules, daily tasks, attendance tracker, and personal notes.

## 2. Core Features (MVP)
* **User Authentication:** Secure sign up, login, and password management.
* **Timetable Management:** Add, view, and manage class schedule by day and time.
* **Task / Assignment Tracker:** Add tasks, set due dates, mark as complete, and track progress.
* **Attendance Tracker:** Mark attendance and view overall attendance percentage (Alert if < 75%).
* **Notice Board:** Admin can post notices, students can view updates.
* **Notes Section:** Create, edit, and organize personal study notes.
